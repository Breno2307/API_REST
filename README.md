# API Petshop - Sistema de Gerenciamento
trabalho da matéria backend

## Sobre o Projeto
API RESTful desenvolvida em Node.js para gerenciamento completo de um petshop. O sistema permite agendamento de serviços, controle de produtos, cadastro de usuários e autenticação segura.
